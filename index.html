Habits Tracker

Un progetto per produrre un semplice habits tracker, che consente di gestire le abitudini giornaliere in modo semplice ed intuitivo, utilizzando uno stack moderno, stabile, scalabile.

Tecnologie chiave

Tecnologie moderne e consolidate per garantire efficienza e modernità

Next.js

Framework React Fullstack per applicazioni scalabili e performanti

Prisma

ORM per query robuste, sviluppo veloce, gestione dati avanzata e affidabilità

MySQL

Database relazionale, affidabile e popolare

Interfacce Utente

Design pulito e funzionalità intuitive per una user experience eccellente.

Autenticazione

Autenticazione sicura, semplice e veloce sia con credenziali, sia con provider esterni (Google, GitHub)

Gestione giornaliera

Spunta le abitudini completate giorno per giorno e in più visualizza tutto lo storico in maniera semplice e veloce

Gestione categorie

Gestisci le categorie in modo semplice e personalizzato, decidendo titolo, colore, icona. Il tutto in modo super semplice e intuitivo

Gestione abitudini

Gestisci le abitudini in modo semplice e personalizzato, decidendo se visualizzare o nascondere le abitudini disattivate. Il tutto in modo super semplice e intuitivo

Gestione profilo

Gestisci tutto ciò che riguarda il tuo profilo, potendolo creare, modificare ed eliminare in qualsiasi momento. Inoltre, gestisci comodamente le preferenze di lingua e di tema

Aggiornamento preferenze

Aggiorna in modo semplice, veloce e intuitivo le tue preferenze di tema e di lingua

Librerie esterne

Un ecosistema tecnologico solido e all'avanguardia.

Tailwind CSS

Utility-first CSS framework altamente personalizzabile

TweakCN

Strumento moderno per la gestione di classi CSS dinamiche

Shadcn UI

Componenti UI React leggeri e altamente personalizzabili

AuthJS

Libreria di autenticazione ottimizzata per Next.js, modulare, comoda e sicura

React Email

Componenti React per costruire template email compatibili

Next-Intl

Internazionalizzazione i18n ottimizzata per Next.js
